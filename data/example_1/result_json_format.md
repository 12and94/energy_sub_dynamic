# 结果 JSON 格式说明

该文件描述 solver 输出的 `_result.json` 文件结构与字段含义。

## 顶层结构

```json
{
  "basis": [[...], ...],
  "eigvals": [...],
  "Kr": [[...], ...],
  "young": 100000.0,
  "poisson": 0.3,
  "vertex_mass": [...]
}
```

## 字段说明

### basis
- 类型：二维数组（矩阵）
- 维度：`[ndof_full, modes]`
- 含义：模态基向量（已回填到完整自由度空间）。

### eigvals
- 类型：一维数组（向量）
- 维度：`[modes]`
- 含义：对应模态的特征值（从小到大）。

### Kr
- 类型：二维数组（矩阵）
- 维度：`[modes, modes]`
- 含义：模态空间中的刚度矩阵 `K_r = B^T K B`。

### young
- 类型：数值
- 含义：杨氏模量。

### poisson
- 类型：数值
- 含义：泊松比。

### vertex_mass
- 类型：一维数组（向量）
- 维度：`[vertex_count]`
- 含义：每个顶点的 lumped 质量（每个四面体的质量按体积平均分配到 4 个顶点）。
